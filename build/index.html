Generation configuration (2/2)

Click here to show/hide the explanations

You can set here the parameters for the cut. All values are in inches.

You have access to a visualized version of the cut. It displays how the material (board) will be cut. The surrounding red rectangle show the size of material needed (on the sizes appears the dimension in inch).

It is possible to chose how the different parts will be placed on the board or if they should be cut in two different boards.

  • Feed rate: the feed rate during the cut
  • Board thickness: the board thickness
  • Bit length: the bit length
  • Bit width: the bit width
